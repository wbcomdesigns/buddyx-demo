BuddyX Demo Importer simplifies the process of setting up your BuddyX-powered site with one-click demo imports.

== Description ==
The **BuddyX Demo Importer** plugin allows users to quickly and easily import demo content for the BuddyX Theme. With just one click, you can import pages, posts, widgets, theme settings, and more to replicate the demo site in minutes. This tool is designed to make the setup process seamless, saving time and effort for both beginners and professionals.

**Features:**
* One-click demo import for BuddyX Theme.
* Imports demo content including posts, pages, menus, and widgets.
* Fully compatible with BuddyPress, LearnDash, LifterLMS, WooCommerce, and other supported plugins.
* Supports importing demos for eCommerce, education, community, and charity sites.
* Error notices and troubleshooting for missing dependencies like BuddyX Theme.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory or install it via the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Appearance > Import Demo Data** to access the demo importer.

== Frequently Asked Questions ==

= What are the requirements for using this plugin? =
The BuddyX Demo Importer requires the BuddyX Theme to be installed and activated.

= Can I use this plugin with other themes? =
No, this plugin is specifically designed for use with the BuddyX Theme.

= What happens if I import a demo multiple times? =
Importing a demo multiple times may result in duplicate content. It is recommended to use a fresh WordPress installation.

= Is this plugin compatible with multisite? =
Yes, BuddyX Demo Importer supports WordPress multisite installations.
